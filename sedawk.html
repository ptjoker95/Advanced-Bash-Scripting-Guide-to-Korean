부록 C. Sed 와 Awk 초급

목차
C.1. Sed
C.2. Awk

다음은 텍스트 프로세싱 유틸리티인 sed 와 awk 의 매우 간단한 소개입니다. 여기서는 몇몇 기본적인 명령어들만 다를 것이나, 쉘 스크립트 내에서 Sed 와 Awk 를 간단히 다루기에는 충분할 것입니다.

sed: 비대화형 (a non-interactive) 모드의 텍스트 파일 에디터

awk: C 형태의 문법을 갖는 필드 단위의 패턴 (field-oriented pattern) 처리 언어

이런 차이점에도 불구하고 이 둘은 비슷한 실행 문법을 갖는데, 정규표현식 을 쓰고, 기본입력 으로부터 입력을 받고, 기본출력 으로 출력을 합니다. 이것들은 잘 작동하는 UNIX 도구이며, 서로같이도 잘 써집니다. 하나의 출력은 다른 쪽에 파이프될 수있고, 이것을 잘 쓰면 스크립트를 Perl 의 능력과 거의 비슷한 수준으로 끌어올릴 수 있습니다.

한가지 중요한 차이점은 쉘 스크립트는 Sed에 쉽게 인수를 넘겨줄 수 있지만, Awk에는 좀 까다롭다는 것입니다. ( 예제 36-5 와 예제 28-2 를 참고하십시요).
